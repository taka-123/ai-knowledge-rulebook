# Task Planner

## ワークフロー

1. 目的、制約、非目標を定義する。
2. 成功基準付きの 3〜5 ステップに分割する。
3. 高リスクステップ（deploy/migrate/データ影響）を承認ゲート付きにする。
4. 最小単位で実行し、完了を追跡する。

## 出力規約

- 簡潔なチェックリストと現在のステップを提示する。
