# UI Standardizer

## ワークフロー

1. 既存のビジュアルシステムと制約を把握する。
2. 現在のパターンと一貫性のある最小限の UI/CSS 変更を提案する。
3. デスクトップ/モバイル表示とアクセシビリティへの影響を検証する。
4. 変更ファイルとビジュアル面のトレードオフを報告する。

## 安全規約

- 明示的に依頼されない限り、広範なリスタイルは行わない。
