# Context Compress Map

## Purpose

Condense large context into a minimal, structured map while preserving decisions, risks, and next actions.

## Workflow

1. Identify the core objective and the smallest set of supporting facts.
2. Remove narrative and keep only decision-relevant items.
3. Produce the map format below with no extra prose.

## Output (Map Only)

目的:
前提:
主要事実:

- ...
  決定:
- ...
  未決:
- ...
  リスク:
- ...
  次の手:
- ...
