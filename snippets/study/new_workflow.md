# AI駆動型学習ワークフロー v2（10ファイル体制）

> **目的**: NotebookLM + Gemini 3 Pro の連携を最大活用し、知識を「実務の武器」に変える
>
> **設計思想**: 認知負荷を最小化し、学習の定着度を最大化する

---

| 分類               | ファイル                          | 用途            | 使用タイミング      |
| :----------------- | :-------------------------------- | :-------------- | :------------------ |

## ステップ0: 環境準備（学習開始前）

### 0-1. 資料の電子化

- **ツール**: GoodNotes / Scanner Pro
- **目的**: 参考書をPDF化し、OCR処理で検索可能にする

### 0-2. NotebookLMへのアップロード

- **ツール**: NotebookLM
- **重要**: **章ごとにソースを分割**してアップロード（検索精度と参照の明確化のため）

### 0-3. Gemカスタム指示の設定

- **設定ファイル**: [GemNLMアナリスト・メンター.md](prompt/GemNLMアナリスト・メンター.md)
- **目的**: Gemini 3 Proに「中堅SE向けメンター」の視点を常時維持させる

---

## ステップI: インプット・戦略設計フェーズ

学習開始時に**全体像と戦略**を把握する。

### I-1. 視覚・聴覚インプット（NotebookLM機能）

#### 動画解説（基本：メイン）

- **ツール**: NotebookLM Video Overview
- **プロンプト**: [NLM動画解説プロンプト3.md](prompt/NLM動画解説プロンプト3.md)
- **目的**: 構造化された視覚インプットで全体像を掴む。時間がある時は必ず実施。

#### スライド解説（時短オプション：準メイン）

- **ツール**: NotebookLM Slide Deck
- **プロンプト**: [NLMスライド資料プロンプト.yml](prompt/NLMスライド資料プロンプト.yml)
- **目的**: 動画を見る時間がない時に、自分のペースで高速に要点を把握する（タイパ重視）。

#### 音声解説（ながらオプション：サブ）

- **ツール**: NotebookLM Audio Overview
- **プロンプト**: [NLM音声解説プロンプト.md](prompt/NLM音声解説プロンプト.md)
- **目的**: 移動中などの隙間時間にインプット（※プロンプトで「画面を見ない前提」に設定済み）

### I-2. 戦略的全体分析（Canvas）

- **モデル**: Gemini 3 Pro
- **ツール**: Canvas
- **プロンプト**: [GemNLM分析（Canvas）.md](prompt/GemNLM分析（Canvas）.md)
- **目的**: 本格的に読み始める前に「手札・教養・スキップ」の戦略地図を手に入れる

**出力イメージ**:

```markdown
### 第N章: [章タイトル]

#### 実務翻訳タイトル: 「今の仕事で言うと、要するに何の話？」

- 🃏 応用力を倍増させる手札: [攻めの知識]
- 🛡️ SEとしての嗜み: [守りの常識]
- ⏭️ 真のスキップ基準: [切り捨てていい詳細]
```

## ステップII: 集中学習フェーズ

GoodNotesで参考書を読み進めながら、AIを活用して理解を深める。

### II-1. 通常の読書とチャット

- **ツール**: Gemini 3 Pro（カスタム指示設定済み）
- **方法**: 疑問が浮かんだらその場でチャット
- **ポイント**: カスタム指示により、ソクラテス式の対話が自然に発生

### II-2. 難所突破（深掘りモード）

通常のチャットでは腹落ちしない概念に遭遇した時に使用。

- **プロンプト**: [難しい概念を解説.md](prompt/難しい概念を解説.md)
- **目的**: 「歴史的背景」「トレードオフ」「物理的イメージ」の3層で強制的に深掘り

**使用基準**:

- ❌ 「なんとなくわかった」で済ませてしまいそうな時
- ❌ 用語の意味はわかるが「なぜそうなのか」が腑に落ちない時
- ❌ 実務でどう使うかイメージできない時

### II-3. 視覚化（オプション）

#### インタラクティブHTML図解

- **プロンプト**: [概念の図解化.md](prompt/概念の図解化.md)
- **目的**: 関係性や構造を視覚化。ブラウザでインタラクティブに操作可能

#### インフォグラフィック（画像）

- **ツール**: NotebookLM Infographic
- **プロンプト**: [NLMインフォグラフィック.md](prompt/NLMインフォグラフィック.md)
- **設定**: レイアウト「横向き」、詳細レベル「詳細（ベータ版）」
- **目的**: GoodNotesに直接貼り付けて保存する用

## ステップIII: 武器化・定着フェーズ

学んだ知識を「実務で使える手札」に昇華させる。

### III-1. ガイド付き深掘り学習

- **モデル**: Gemini 3 Pro
- **ツール**: ガイド付き学習
- **プロンプト**: [GemNLM質問（ガイド付き学習）.md](prompt/GemNLM質問（ガイド付き学習）.md)
- **目的**: 知識を実務でどう活用できるかをAIと壁打ちし、「使える手札」へと昇華させる

### III-2. 武器化完了チェック（セッション終了時）

上記プロンプトの末尾に組み込まれた「3つの問い」に、**NotebookLMを参照せずに**回答する。

1. **実務接続**: 今日学んだ〇〇を、明日の仕事で使うとしたら、どの場面で何を判断する根拠になるか？
2. **Before/After**: この知識がなかった頃の自分は、その場面でどう判断していたか？
3. **30秒説明**: 同僚に30秒で説明するなら、どう伝えるか？

**この3問に答えられれば、その知識は「武器化完了」**

## ステップIV: 知識資産化フェーズ（オプション）

学習成果を美しいPDF形式で保存する。

### IV-1. HTML/PDFへの整形

- **プロンプト**: [テキストをHTML資料化.md](prompt/テキストをHTML資料化.md)
- **目的**: 対話ログやメモを美しいHTMLに変換し、ブラウザでPDF保存
- **用途**: GoodNotesに取り込み、検索可能な知識資産として蓄積

## クイックリファレンス

### 日常の学習フロー（最小構成）

```
1. NotebookLM動画解説 → 全体像把握（5-10分）
2. Canvas分析 → 戦略地図作成（10分）
3. GoodNotes読書 + Geminiチャット（メイン学習）
4. 武器化チェック → 3つの問いに回答（5分）
```

### 「困った時」の対処法

| 状況                           | 使うプロンプト               |
| :----------------------------- | :--------------------------- |
| 用語はわかるが本質がわからない | `難しい概念を解説.md`        |
| 関係性がごちゃごちゃ           | `概念の図解化.md`            |
| 綺麗に保存したい               | `テキストをHTML資料化.md`    |
| 一枚絵で俯瞰したい             | `NLMインフォグラフィック.md` |
