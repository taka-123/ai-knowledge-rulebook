# 出力例

## 全体の要約

この記事では、デジタルヘルス領域におけるウェルネス産業の拡大を取り上げ、
健康寿命の延伸や医療のDXが求められる中、テクノロジーを駆使して「健康」を支えるスタートアップ7社の事例を紹介しています。
それぞれが医療・介護・フィットネスなど多岐にわたる分野で新しい価値を創造しており、ウェルネス産業の未来を描いています。

## ポイント

1. **Ubie**: 医師の思考をアルゴリズム化し、問診プロセスの効率化や適切な医療案内を実現するプラットフォームを構築。大規模データ活用に挑む
2. **Rehab for JAPAN**: 高齢者支援に特化したデジタルプラットフォームを構築し、介護現場の効率化やリハビリ支援を推進
3. **CureApp**: デジタル治療（DTx）を活用した治療アプリを提供し、生活習慣病や禁煙治療の新しいアプローチを提案
