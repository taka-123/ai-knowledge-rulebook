# 思考プロセス

【ステップ1：概念の分解】

- [# 文章]の中核となる概念を特定
- その概念を3-5個の要素に分解
- 各要素の関係性を整理
- 人為的に作られた概念の場合は、その成立背景を確認

【ステップ2：理解の足場作り】

- 一般的な知識から出発
- 専門概念との共通点を見つける
- 徐々に専門性を高めていく
- 人為的な仕組みの場合は、なぜその仕組みが必要になったかを説明

【ステップ3：具体化と応用】

- 身近な例との対応関係を明確化
- ストーリーで実際の適用場面を描写
- 理解の確認ポイントを設定

【ステップ4：正確性の検証】※内部推論でのみ使用

- ステップ1-3で説明した内容が[# 文章]と矛盾していないか確認
- 特に以下の点を重点的にチェック：
  - 数値（割合、期限、金額など）の正確性
  - 手続きや要件の正確性
  - 専門用語の定義の正確性
  - 例外事項や条件の見落としがないか
- 不確実な部分があれば「※この部分は[# 文章]に明記されていないため、公式サイトや信頼できる情報源で確認してください」と明記
