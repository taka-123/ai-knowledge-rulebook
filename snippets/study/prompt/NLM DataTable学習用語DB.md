# 目的

技術書やドキュメントから「重要な専門用語・概念・コマンド」を抽出し、復習および暗記カード作成用のデータベース（Data Table）を作成する。

# Constraints（制約）

- ソース内の記述に基づいて正確に抽出すること。
- 推測による補完は行わず、明記がない場合は「-」とすること。
- 定義は「一言で」簡潔にまとめること（長文は避ける）。

# Output Format (Columns)

以下の5列でテーブルを作成してください。

1. **用語/コマンド名**
   - 英語表記がある場合は ( ) で併記
   - 例: 依存性の注入 (Dependency Injection)

2. **カテゴリ**
   - 例: 概念, ツール, コマンド, 設定値, 人物

3. **簡潔な定義・役割**
   - 30文字〜50文字程度で「それは何か」を説明

4. **メリット/使用場面**
   - なぜそれを使うのか、いつ使うのか

5. **関連用語/対義語**
   - 関連する他の用語があれば記載

# Instruction

アップロードされた資料全体（または指定された範囲）から、学習者が覚えるべき重要なキーワードを網羅的に抽出し、上記のフォーマットで表を作成してください。
