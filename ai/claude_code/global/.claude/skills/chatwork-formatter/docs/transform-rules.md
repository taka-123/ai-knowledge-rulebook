# ChatWork 記法対応表

記法の対応表のみを扱う。`[info]` を使うかどうかの判定と、変換で形式を増幅しない方針は `SKILL.md` に従う。

## タグの使用条件（重要）

- `[title]` は **`[info]` 内でのみ** 使用可。単独では使わない。
- `[info]` は単独で使用可。`[title]` を伴わなくても構わない。
- `[code]` `[hr]` は単独で使用可。

## 見出し変換

| 状況                    | Markdown | ChatWork                                   |
| ----------------------- | -------- | ------------------------------------------ |
| `[info]` で囲む場合     | `# H1`   | `[info][title]H1[/title]...内容...[/info]` |
| `[info]` を使わない場合 | `# H1`   | `【H1】`（前後に空行）                     |
| どちらでも              | `## H2`  | `▼ H2`（前後に空行）                       |
| どちらでも              | `### H3` | `■ H3`                                     |

## その他の変換

| Markdown                 | ChatWork                                   |
| ------------------------ | ------------------------------------------ |
| `- item` / `* item`      | `・item`                                   |
| `  - subitem`（ネスト）  | `　・subitem`（全角スペース+中点）         |
| `1. item`                | そのまま保持                               |
| `**bold**`               | `**` を除去、テキストのみ                  |
| `*italic*`               | `*` を除去、テキストのみ                   |
| `` `inline code` ``      | バッククォートを除去、テキストのみ         |
| ` ```コードブロック``` ` | `[code]...[/code]`                         |
| `---`（水平線）          | `[hr]`                                     |
| `[text](url)`            | `text ( url )`                             |
| `> 引用`                 | `　` +テキスト（全角スペースでインデント） |

## 優先ルール

- ChatWork で表示できない記法は削除・平文化する（崩れたまま渡さない）。
- 絵文字の記法は `(bow)` `(roger)` `(cracker)` など（追加の可否は `SKILL.md` に従う）。
- `[info]` の採否は `SKILL.md` のメッセージ種別判定に従う（デフォルトはプレーンテキスト）。
